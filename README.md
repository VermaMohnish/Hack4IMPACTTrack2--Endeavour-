Problem Statement :
Urban Disaster Response in India is Reactive, Not Proactive.

When disasters like floods strike Bhubaneswar, emergency services respond only
after damage occurs. There is no system that predicts which specific localit are at risk, pre-
positions rescue assets, or coordinates response in real time.
